Name Type Attributes Description

Environment key-value pairs.

Explicitly set the value of argv[0] sent to the child process.

Child's stdio configuration

Prepare child to run independently of its parent process. (Operating system dependant behavior)

Hide the subprocess console window that would normally be created on Windows systems.

timeout number <optional>

Timeout, in milliseconds the maximum amount of time the process is allowed to run.

The signal value to be used when the spawned process will be killed by timeout or abort signal.
